Changelog

1.7.1 - 16 Feb 2010

  • Fixed bug introduced in 1.7.0 where option changes wouldn't take effect until restart.

1.7.0 - 07 Jan 2010

  • Free version (GPL).

1.6.0 - 21 Feb 2008

  • Enabled mapping to/from Browser, Volume, Media and Launch keys.

1.5.0 - 05 Jan 2008

  • Added check-for-updates functionality.

1.4.0 - 10 Dec 2007

  • It is now possible to output Shift, Ctrl, Alt and Win in combination with a key. E.g. Space+X -> Ctrl+Y.

  • Added "Beep for mistakes" option in training mode.

  • Installer backs up old settings file.

  • When editing an existing key binding, the "Key to generate" box is focused by default.

1.3.1 - 23 Oct 2007

  • Fixed a bug where key presses would sometimes pass through unchanged when they shouldn't.

  • Improved installer behaviour on Windows Vista.

1.3.0 - 12 Sep 2007

  • Made the activation key configurable. (Previously space bar only.)

1.2.2 - 13 Jul 2007

  • Fixed bug where generated Escape events would sometimes be ignored by Virtual PC and VMware.

1.2.1 - 08 Jun 2007

  • Works with full-screen Remote Desktop Connection and VMware.
    (Note that versions of VMware that have the "enhanced virtual keyboard" option must have this option disabled for the host's TouchCursor to work when using the guest OS.)

  • Installer adds a Start Menu shortcut for touchcursor.exe (not usually needed due to "Run at startup" option).

  • Installer sets AppPath for touchcursor.exe.

1.2.0 - 24 May 2007

  • Works with Microsoft Virtual PC and Remote Desktop Connection (but not full-screen RDC yet).

  • Alt+Space now works in a DOS box.

  • Some nag screen tweaks.

1.1.1 - 18 Apr 2007

  • License change.

1.1.0 - 09 Apr 2007

  • Settings stored in a file instead of the registry.

  • Digitally signed installer and application files.

  • Minor dialog text re-wordings.

1.0.6 - 19 Feb 2007

  • The icon in the notification area is optional.

1.0.5 - 8 Feb 2007

  • Bug fix for occasional failure of "Run at startup".

1.0.4 - 2 Feb 2007

  • You can now bind Tab events, using the "..." button on the binding dialog. Escape is now bound in this way too, instead of having non-standard Escape handling on the dialog.

  • Enabled binding of Shift and Ctrl events. Note that the training mode can be pretty unbearable with these bound, unfortunately.

  • When space is released, key-up events are generated for any generated keys that are still held down .

  • Correct, locale-proof mapping is used from OEM key codes to characters. (Previously hard-coded to US keyboard layout.)

  • Added accelerator for F1 help.

1.0.3 - 28 December 2006

  • Support for running from a USB drive.

  • Zip distribution.

1.0.2 - 1 December 2006

  • Works on Windows 2000

  • Uninstall asks for feedback.

1.0.1 - 17 November 2006

  • Trial period changed to 30 days.

1.0.0 - 14 November 2006

  • Initial release.
