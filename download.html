TouchCursorSetup-1.7.1.exe

Most people will want this one, as it includes an uninstall facility. If limited Windows privileges prevent you from using this installer, you can use the zip installation below instead.

(1MB Windows installer.)

TouchCursor-1.7.1.zip

Use this if you want to run TouchCursor from a USB drive, or if you don't want to use the installer for any reason. Simply extract the zip file to the location of your choice and run touchcursor.exe. On removable drives such as USB drives the configuration will be stored in the same folder as the program, allowing you to move it from machine to machine.

(1MB Zip file.)
